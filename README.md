# 👨‍🍳 Flavortown CLI Problem Solver

Este é meu projeto era para o Nibbles do Flavortown (Hack Club), mas não terminei então continuo desenvolvendo ele. Sou iniciante total em programação e desenvolvi este quiz de terminal inteiramente em um smartphone (Android) usando **Termux** e **Acode**.

## 🚀 Como testar
1. No terminal/Termux: `python main.py`
2. Responda às perguntas de lógica e programação.

## 🛠️ Tecnologias
- **Lógica:** Python 3 + JSON
- **Ambiente:** Mobile (Termux/Acode)
- **Landing Page:** Design feito no Figma/Frame
